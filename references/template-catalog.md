# 模板库速查

模板资产位于 `assets/templates/`，联系表位于 `assets/previews/`。选择器读取 `references/template-catalog.json`。

| ID | 风格 | 适合场景 | 建议角色 |
| --- | --- | --- | --- |
| `academic-clean` | 红白学术、结构清楚 | 研究、论文、正式课程 | 主模板 |
| `dark-editorial` | 深灰红、现代编辑感 | 科技、项目、沉稳汇报 | 主模板或版式供体 |
| `campus-photo` | 校园大图、红白清爽 | 校园介绍、课程展示 | 图片供体 |
| `bold-geometry` | 大红几何、强章节感 | 答辩、正式发布 | 章节供体 |
| `deep-red` | 深红沉浸、极简 | 正式演讲、品牌感 | 主模板 |
| `analytical-clean` | 白底分析、图表丰富 | 数据、市场、研究结果 | 数据供体 |
| `defense-nav` | 导航式答辩、表格严谨 | 毕业答辩、项目审查 | 主模板或数据供体 |
| `premium-overlay` | 校园叠色、层次丰富 | 高质量答辩、领导汇报 | 主模板 |
| `coral-data` | 珊瑚红、轻量信息图 | 青年课堂、数据讲解 | 数据供体 |
| `campus-arc` | 校园弧形大图、现代 | 课程汇报、学校主题 | 主模板或图片供体 |
| `organic-wave` | 红色曲线、活跃现代 | 创意课程、校园活动 | 版式供体 |
| `gradient-red` | 红色渐变、柔和现代 | 通用课程、总结汇报 | 主模板或强调供体 |
| `winter-grey` | 冬季灰绿、雪景 | 冬季主题、沉静叙事 | 季节强调供体 |
| `autumn-orange` | 秋季橙棕、摄影感 | 秋季主题、校园人文 | 季节强调供体 |

默认组合：

```text
academic-clean + analytical-clean + campus-photo
```

这组分别提供品牌与学术结构、数据版式、校园摄影。
