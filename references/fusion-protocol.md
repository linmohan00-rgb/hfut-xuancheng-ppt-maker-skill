## 三种模板角色

### Visual anchor

主模板。控制整稿的品牌识别和视觉语法，通常贡献 50%-65% 的视觉影响：

- 页面比例和安全边距。
- 校徽、校名、页眉、页脚和页码。
- 标题、正文、数字和注释字体。
- 主色、背景色、正文色和强调色。
- 封面、目录、章节页和结尾页。

### Layout donor

版式供体。通常贡献 20%-35% 的视觉影响：

- 分栏、网格、时间线、流程、对比和复杂图文布局。
- 对象比例、留白和阅读顺序。
- 不拥有最终颜色、字体、校徽或页脚。

### Accent donor

强调供体。通常贡献 10%-20% 的视觉影响：

- 数据图、信息图、摄影裁切、季节氛围或特殊章节构图。
- 少量辅助色或纹理。
- 不得形成第二套标题或品牌系统。

## 所有权矩阵

| 元素 | 最终所有者 | 供体可保留的内容 |
| --- | --- | --- |
| 页面比例 | Visual anchor | 无 |
| 校徽与校名 | Visual anchor | 无 |
| 页眉、页脚、页码 | Visual anchor | 无 |
| 标题与正文字体 | Visual anchor | 只保留字号层级关系 |
| 主色与背景 | Visual anchor | 只保留明暗关系 |
| 页面网格 | 当前供体 | 几何、留白、对齐 |
| 图表结构 | 当前供体 | 图形类型、排序、标注位置 |
| 图片裁切 | 当前供体 | 构图与比例 |
| 内容与数据 | 用户材料或核验来源 | 无模板示例内容 |

## 融合步骤

1. 从主模板提取四个语义色：`brand`、`ink`、`paper`、`support`。
2. 提取标题、正文、数字和注释字体；缺失字体统一回退到微软雅黑、等线或 Arial。
3. 对版式供体和强调供体逐页执行：
   - 删除重复校徽、校名、页眉、页脚和页码。
   - 将所有标题切换为主模板标题字体、字号和颜色。
   - 将正文切换为主模板正文字体和正文色。
   - 按语义将供体主色映射到 `brand`，灰色映射到 `ink/paper`，少量强调映射到 `support`。
   - 保留布局几何，不保留独立品牌装饰。
4. 统一圆角、描边、阴影、图片蒙版和图标风格。
5. 渲染联系表，检查是否仍能一眼看出三套独立模板；能看出则继续融合。

## 页面抽取规则

- 封面、目录、主要章节页和结尾页优先来自主模板。
- 版式供体优先承担流程、对比、时间线、方法、案例拆解。
- 强调供体优先承担数据、关键结论、大图、季节或情绪转场。
- 连续三页不得使用完全相同的宏观构图。
- 同一供体模板的复杂装饰页不连续使用超过两页。
- 选择器的 `slide_plan` 是起点；内容不匹配时可在同一已选模板内换页，但不得引入第四套模板。

## 兼容性与回退

- 深色主模板配浅色供体：保留供体几何，重做背景和文字对比。
- 季节模板作为主模板：仅当用户明确要求季节气质；否则作为强调供体。
- 数据很多：优先 `analytical-clean`、`coral-data` 或 `premium-overlay`。
- 图片很多：优先 `campus-photo`、`campus-arc`、`organic-wave`。
- 正式答辩：优先 `academic-clean`、`defense-nav`、`premium-overlay`。
- 供体页面字体缺失：不要安装来历不明字体，统一使用主模板字体。
- PowerPoint COM 不可用：用 Presentations 插件导入模板资产并复制真实源页，禁止从空白页重画近似模板。

## QA 阻断项

以下任一情况必须修复：

- 同一页出现两个不同版本的校徽或校名。
- 相邻页面的标题位置、字体或颜色明显跳变。
- 供体模板页脚、二维码、姓名、测试数字或示例文案残留。
- 汇报人、班级、专业、学院、学号或日期沿用了模板默认值，而不是按用户材料填写或留空。
- 非主模板页面仍保留独立配色体系。
- 页面只是把三套模板依次排列，没有语义色和字体映射。
- 图表颜色与主模板无关，或数据单位不一致。
- 文字溢出、低对比、图片拉伸、对象错位。
