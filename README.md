# 合肥工业大学宣城校区 PPT 制作 Skill

这是一个用于制作或改造合肥工业大学宣城校区风格课堂汇报 PPT 的 Codex Skill。它来自一份 PPT 制作记忆整理，重点保留了合工大红白风格、课堂演讲版定位、资料补充、图片去重、排版检查和演讲稿配套等制作逻辑。

## 能做什么

- 根据课程主题、截图、原始 PPT 或素材制作课堂汇报 PPT。
- 将旧 PPT 改造成合肥工业大学/宣城校区风格的演示文稿。
- 按 6-8 分钟课堂展示控制页数，默认约 12-15 页。
- 保留或补充校徽、校名、专业、班级、姓名、课程信息等身份内容。
- 联网补充权威资料和相关图片，避免图片重复使用。
- 生成 PPT 后检查文字、图片、页码、标题层级和版式一致性。
- 可按需附带逐页演讲稿或视频脚本。

## 适用场景

- “帮我做一份合工大风格课堂汇报 PPT”
- “按照这个截图/旧 PPT 改成合肥工业大学宣城校区风格”
- “做一个 7-8 分钟心理课/历史课/工程案例汇报 PPT”
- “套用之前合工大的模板，自己搜图补内容”
- “PPT 不要答辩风，要课堂演讲用”

## 输出风格

- 主色调：深红、白色，少量金色或浅灰辅助。
- 首页体现学校、课程、主题、汇报人等信息。
- 内容适合课堂讲述，不做空泛答辩模板。
- 每页有明确讲述点，文字不过密。
- 图片与主题高度相关，不重复、不拉伸、不遮挡文字。
- 最终交付前要做页面检查，发现重叠、溢出、错位要先修正。

## 安装方式

将仓库克隆到 Codex 的 skills 目录：

```powershell
git clone https://github.com/linmohan00-rgb/hfut-xuancheng-ppt-maker-skill.git "$env:USERPROFILE\.codex\skills\hfut-xuancheng-ppt-maker"
```

安装后，新会话中即可通过 `$hfut-xuancheng-ppt-maker` 调用。

## 输入建议

为了让制作更稳，可以尽量提供：

- 课程名称和汇报主题；
- 姓名、专业、班级、学号或小组信息；
- 演讲时长和页数要求；
- 原始 PPT、截图、模板或参考图片；
- 是否允许联网检索资料；
- 是否需要逐页演讲稿或视频脚本。

如果信息不全，Skill 会使用占位符或合理默认值继续制作，不会因为姓名、班级等小字段缺失而阻塞整体流程。
